# Carbu'Web

Comparateur de prix de carburants en France.

Trouvez les stations-services les moins cheres autour de vous, comparez les prix par region ou departement, et consultez les statistiques nationales. Le site se met a jour quotidiennement a partir des donnees ouvertes du gouvernement, enrichies avec les noms de stations OpenStreetMap.

Fonctionnalites :

- Recherche par adresse, ville ou code postal
- Geolocalisation : stations-services autour de votre position
- Palmares : classement des stations par prix
- Statistiques nationales et regionales
- Favoris et personnalisation (carburants, rayon de recherche)
- Cartographie interactive
